# Terraform Management Console

Useful for managing your cloud infrastructure from UI.
It can add, update, delete resources on your cloud using Terraform with version control.

Tools and Technologies: <br>
Backend: Go lang <br>
Frontend: React JS <br>
Containerization: Docker, Docker Compose <br>
Monitoring: Prometheus, Grafana <br>
